标识符：
	定义:
		java对各种变量、方法和类等要素命名时使用的字符序列称之为标识符
	规则：
		1.标识符由26个大小写英文字母、数字、下划线、美元符号组成
		2.不能以数字开头，不能是关键字
		3.严格区分大小写
		4.可以为任意长度

	命名规范：
		1.包名
			所有字母都小写
		2.类名和接口名
			首字母大写，多个单词组合时，所有单词的首字母大写

		class  AnimalPark {


		}

		3.变量名和方法名
			首字母小写，多个单词组合时，所有单词的首字母大写

			int userAccount=10;
	
			public void userLogin(){

			}

		4.常量名
			所有字母全部大写，多个单词之间，用_隔开
			final int USER_ACCOUNT=20;

		遵守命名规范，可以提高代码的阅读性、可读性

关键字(保留字)：
	是指java中自带的用于标识数据类型名，或程序结构名等的标识符
	
转义字符：
	\n			换行
	\b			退格
	\t			水平制表
